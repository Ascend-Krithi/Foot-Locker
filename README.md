# Foot Locker Automation Framework

## Overview
Enterprise-grade Selenium TestNG automation framework for Foot Locker web application.

## Tech Stack
- Java 17
- Selenium 4.21+
- TestNG 7.10+
- Maven 3.9+
- Extent Reports 5.1.1

## Setup
1. Clone repository
2. Install Java 17
3. Run: `mvn clean test`

## CI/CD
GitHub Actions workflow runs on every push/PR to main/develop branches.

## Reports
- Extent Reports: `test-output/ExtentReport.html`
- Screenshots: `test-output/screenshots/`
