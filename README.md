# Foot Locker Selenium TestNG Automation Framework

This project is a Java Selenium TestNG automation framework for the Foot Locker web application.

## Features
- 7 TestNG test cases mapped to TestRail
- Strict locator fallback policy
- ExtentReports integration
- Headless Chrome execution
- Maven build

## Prerequisites
- Java 17+
- Maven 3.8+

## Run Tests
```bash
mvn clean test
```

## Test Results
- HTML report: `test-output/ExtentReport.html`
